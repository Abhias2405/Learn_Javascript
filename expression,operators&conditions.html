JavaScript Expressions and Conditions

1. Expressions

An expression is any valid unit of code that resolves to a value.

Examples:


// Arithmetic expression
let sum = 5 + 3; // Resolves to 8

// String concatenation
let message = "Hello, " + "world!"; // Resolves to "Hello, world!"

// Function call expression
let square = Math.pow(2, 3); // Resolves to 8

Expressions are often used with operators to manipulate data.

2. Operators

JavaScript provides several types of operators to perform operations on variables and values.

1. Arithmetic Operators

Used to perform mathematical operations.

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (Remainder)
  • **: Exponentiation

// Example
let a = 10, b = 3;
console.log(a + b); // 13
console.log(a % b); // 1

2. Assignment Operators

Used to assign values to variables.

  • =: Assign
  • +=: Add and assign
  • -=: Subtract and assign
  • *=: Multiply and assign
  • /=: Divide and assign

// Example
let c = 5;
c += 3; // Equivalent to c = c + 3
console.log(c); // 8

3. Comparison Operators

Used to compare two values.

  • ==: Equal (abstract equality)
  • ===: Strict equal
  • !=: Not equal
  • !==: Strict not equal
  • >: Greater than
  • <: Less than

// Example
let x = 10, y = "10";
console.log(x == y); // true (abstract equality)
console.log(x === y); // false (strict equality)

4. Logical Operators

Used to perform logical operations.

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

// Example
let isAdult = true, hasID = false;
console.log(isAdult && hasID); // false
console.log(isAdult || hasID); // true

5. Comments

Comments are used to explain code or make it more readable.

  • //: Single-line comment
  • /* */: Multi-line comment

// This is a single-line comment
let name = "John"; /* This is a multi-line comment */
3. Conditional Statements

Conditional statements control the flow of execution based on a condition.

1. If Statement


// Example
let age = 18;
if (age >= 18) {
    console.log("You are eligible to vote.");
}

2. If-Else Statement


// Example
let isRainy = true;
if (isRainy) {
    console.log("Take an umbrella.");
} else {
    console.log("Enjoy the sunshine.");
}

3. If-Else If-Else Statement


// Example
let marks = 85;
if (marks >= 90) {
    console.log("Grade: A");
} else if (marks >= 75) {
    console.log("Grade: B");
} else {
    console.log("Grade: C");
}

4. Ternary Operator

A shorthand for if-else.


// Example
let isWeekend = false;
let activity = isWeekend ? "Relax at home" : "Go to work";
console.log(activity); // Output: Go to work
