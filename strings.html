JavaScript Strings

1. What is a String?

A string is a sequence of characters enclosed in quotes (single, double, or backticks). Strings are used to represent text in JavaScript.


// Example:
let singleQuote = 'Hello';
let doubleQuote = "World";
let templateLiteral = `Hello World`;

console.log(singleQuote); // Output: Hello
console.log(doubleQuote); // Output: World
console.log(templateLiteral); // Output: Hello World
2. Template Literals

Template literals allow embedded expressions and multi-line strings using backticks (`) instead of quotes.

Features:

  • String Interpolation (inserting variables): ${expression}
  • Multi-line strings

// Example: String Interpolation
let name = "Abhi";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;

console.log(greeting);
// Output: Hello, my name is Abhi and I am 25 years old.

// Example: Multi-line Strings
let multiLine = `This is a
multi-line
string.`;

console.log(multiLine);
// Output:
// This is a
// multi-line
// string.
3. Escape Sequence Characters

Escape sequences allow special characters to be included in strings.

Common Escape Sequences:

  • \n: New line
  • \t: Tab
  • \\: Backslash
  • \': Single quote
  • \": Double quote

// Example:
let text = "This is a line.\nThis is another line.\tTabbed in.";
console.log(text);
// Output:
// This is a line.
// This is another line.   Tabbed in.
4. String Properties and Methods

1. Properties

  • length: Returns the number of characters in the string.

// Example:
let name = "JavaScript";
console.log(name.length); // Output: 10

2. Methods

  • toUpperCase(): Converts all characters to uppercase.
  • toLowerCase(): Converts all characters to lowercase.
  • slice(start, end): Extracts a section of a string.
  • replace(oldValue, newValue): Replaces a part of the string.
  • concat(): Combines two or more strings.
  • trim(): Removes whitespace from both ends of the string.

// Example:
let str = "  Hello JavaScript!  ";
console.log(str.toUpperCase()); // Output: HELLO JAVASCRIPT!
console.log(str.toLowerCase()); // Output: hello javascript!
console.log(str.slice(2, 7));   // Output: Hello
console.log(str.replace("JavaScript", "World")); // Output:   Hello World!  
console.log(str.concat(" It's fun!")); // Output:   Hello JavaScript!   It's fun!
console.log(str.trim());        // Output: Hello JavaScript!
5. Strings are Immutable

In JavaScript, strings are immutable, meaning their content cannot be changed after they are created. Any operation that seems to modify a string actually creates a new string.


// Example:
let original = "Hello";
let modified = original.replace("H", "J");

console.log(original); // Output: Hello
console.log(modified); // Output: Jello
6. String Indexing

Strings can be accessed using their index, where the first character is at index 0.


// Example:
let name = "Abhi";
console.log(name[0]); // Output: A
console.log(name[1]); // Output: b
