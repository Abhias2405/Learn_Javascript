JavaScript Arrays

1. What is an Array?

An array is a special variable in JavaScript used to store multiple values in a single variable. Arrays are zero-indexed, meaning the first element is at index 0.

// Example:
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Output: Apple
console.log(fruits[1]); // Output: Banana
2. Accessing and Modifying Values

You can access an array's elements using their index, and you can modify them by assigning a new value.

// Example:
let numbers = [10, 20, 30];
console.log(numbers[1]); // Output: 20

// Changing the value at index 1
numbers[1] = 25;
console.log(numbers); // Output: [10, 25, 30]
3. Finding Array Length

The length property returns the number of elements in an array.

// Example:
let languages = ["JavaScript", "Python", "Java"];
console.log(languages.length); // Output: 3
4. Common Array Methods

1. Converting Arrays

  • toString(): Converts the array to a comma-separated string.
  • join(): Joins all elements into a string with a specified separator.
// Example:
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits.toString()); // Output: Apple,Banana,Cherry
console.log(fruits.join(" - ")); // Output: Apple - Banana - Cherry

2. Modifying Arrays

  • push(): Adds an element to the end of the array.
  • pop(): Removes the last element.
  • shift(): Removes the first element.
  • unshift(): Adds an element to the beginning of the array.
  • splice(): Adds or removes elements from a specific index.
  • slice(): Returns a new array with selected elements.
// Example:
let numbers = [1, 2, 3, 4];
numbers.push(5); // [1, 2, 3, 4, 5]
numbers.pop();   // [1, 2, 3, 4]
numbers.shift(); // [2, 3, 4]
numbers.unshift(0); // [0, 2, 3, 4]

// Using splice:
numbers.splice(1, 2, 8, 9); // [0, 8, 9, 4]

// Using slice:
let newNumbers = numbers.slice(1, 3); // [8, 9]

3. Sorting and Reversing

  • sort(): Sorts the array alphabetically or by custom logic.
  • reverse(): Reverses the order of the array.
// Example:
let nums = [3, 1, 4, 2];
nums.sort(); // [1, 2, 3, 4]
nums.reverse(); // [4, 3, 2, 1]
5. Looping Through Arrays

1. forEach()

Executes a function for each element in the array.

// Example:
let nums = [10, 20, 30];
nums.forEach(num => console.log(num));
// Output:
// 10
// 20
// 30

2. map()

Creates a new array by applying a function to each element.

// Example:
let nums = [1, 2, 3];
let squared = nums.map(num => num * num);
console.log(squared); // Output: [1, 4, 9]

3. filter()

Creates a new array with elements that pass a test.

// Example:
let nums = [10, 15, 20];
let filtered = nums.filter(num => num > 12);
console.log(filtered); // Output: [15, 20]

4. reduce()

Reduces the array to a single value by applying a function.

// Example:
let nums = [1, 2, 3, 4];
let sum = nums.reduce((total, num) => total + num, 0);
console.log(sum); // Output: 10
