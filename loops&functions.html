JavaScript Loops and Functions

1. Loops

Loops are used to execute a block of code repeatedly, either a fixed number of times or until a condition is met.

Types of Loops:

1. For Loop

Used when the number of iterations is known beforehand.


// Example: Print numbers from 1 to 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

2. For...in Loop

Used to iterate over the properties of an object.


// Example: Iterating over object properties
const user = { name: "John", age: 30, country: "USA" };
for (let key in user) {
    console.log(key + ": " + user[key]);
}

3. For...of Loop

Used to iterate over the values of iterable objects like arrays, strings, etc.


// Example: Iterating over an array
const fruits = ["apple", "banana", "cherry"];
for (let fruit of fruits) {
    console.log(fruit);
}

4. While Loop

Executes a block of code as long as the condition is true.


// Example: Print numbers from 1 to 5
let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}

5. Do...while Loop

Executes the code block at least once, and then continues while the condition is true.


// Example: Print numbers from 1 to 5
let j = 1;
do {
    console.log(j);
    j++;
} while (j <= 5);
2. Functions

Functions are reusable blocks of code that perform a specific task.

1. Basic Function


// Example: A simple function
function greet() {
    console.log("Hello, world!");
}
greet(); // Output: Hello, world!

2. Function with Parameters

Functions can accept inputs, known as parameters, to perform operations.


// Example: Function with parameters
function sum(a, b) {
    return a + b;
}
console.log(sum(5, 3)); // Output: 8

3. Arrow Functions

A concise way to define functions using the => syntax.


// Example: Arrow function
const multiply = (a, b) => a * b;
console.log(multiply(4, 5)); // Output: 20
3. Arguments in JavaScript

In JavaScript, the arguments object is an array-like object accessible inside functions that represents the arguments passed to the function.


// Example: Using the arguments object
function showArguments() {
    for (let i = 0; i < arguments.length; i++) {
        console.log(arguments[i]);
    }
}
showArguments(1, "hello", true);
// Output:
// 1
// hello
// true

Rest Parameters

Instead of using arguments, modern JavaScript allows you to use rest parameters to handle a variable number of arguments.


// Example: Using rest parameters
function showRestArgs(...args) {
    console.log(args); // Output: [1, "hello", true]
}
showRestArgs(1, "hello", true);
